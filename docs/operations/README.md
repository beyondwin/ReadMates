# Operations

## 하위 문서

- [Runbooks](runbooks/README.md) — 반복 운영 절차, 배포 attempt, 읽기 전용 진단, 배포 후 관찰.
- [Post-mortems](postmortems/README.md) — 발생한 incident의 회고 기록.
- [Observability](observability/README.md) — 메트릭, 대시보드, 알림 룰, SLO.
